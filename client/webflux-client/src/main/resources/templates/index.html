Example use Spring Webflux
1. Go to the "Genereted Data" page to see the action using the generated data from the resource: http://localhost:9081/rest/generateddata/fuel/event
2. Go to the "Mongo Data" page to see the action using the Mongo database data from the resource: http://localhost:9081/rest/mongodata/fuel/event
Before you go to the "Mongo Data" page, add several data documents in the Postman as follows:

[ { "name": "PB95", "price": 4.11, "time": {{$timestamp}} },
{ "name": "PB98", "price": 4.32, "time": {{$timestamp}} },
{ "name": "ON", "price": 4.55, "time": {{$timestamp}} },
{ "name": "LPG", "price": 2.36, "time": {{$timestamp}} } ]

Add different prices for different values
resource: localhost:9081/rest/fuel/addAll

If you have already added several documents, go to the "Mongo Data" page

After going to the page, wait a moment for the chart to be drawn and then add further documents using the resource: localhost:9081/rest/fuel/addAll
